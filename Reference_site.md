# Reference Site Analysis

## 1. Google Antigravity (Frontend Use Cases)
**URL**: https://antigravity.google/use-cases/frontend

### 🎨 UX/UI 강점
*   **Minimalism & Trust**: 극도로 절제된 레이아웃과 넉넉한 여백(White Space)을 통해 콘텐츠 집중도를 극대화했습니다. "High-end Corporate Tech" 감성을 전달합니다.
*   **Typography Hierarchy**: Google Sans 폰트를 활용하여 제목과 본문의 위계가 명확하며, 가독성이 뛰어납니다.
*   **Consistent Design Language**: 구글의 브랜드 아이덴티티를 유지하면서도 실험적인 웹 기술을 세련되게 포장했습니다.

### ⚡ 인터랙션 요소 (Interaction)
*   **Scroll Reveal**: 스크롤 시 콘텐츠가 부드럽게 페이드인(Fade-in) 되며 등장하여 정보 과부하를 방지합니다.
*   **Micro-interactions**: 버튼 호버 시 미세한 그림자 변화와 부드러운 색상 전이(Transition)가 고급스러움을 더합니다.

### 🌈 컬러 시스템
| 구분 | 색상(Color) | Hex Code (추정) | 비고 |
| :--- | :--- | :--- | :--- |
| **Primary** | **Google Blue** | `#4285F4` | 신뢰, 기술, 지성 |
| **Text** | **Deep Charcoal** | `#202124` | (Neutral 900) 완벽한 블랙보다 눈이 편안함 |
| **Background** | **Pure White** | `#FFFFFF` | 깨끗하고 넓은 공간감 |

## 2. Osmo Supply
**URL**: https://www.osmo.supply/

### 🎨 UX/UI 강점
*   **Industrial & Raw Vibe**: 그리드 라인을 그대로 노출하고 대담한 타이포그래피를 사용하여 설계도나 공장 같은 "Raw"한 느낌을 줍니다. 개발자 타겟의 힙한 감성입니다.
*   **Dynamic Layout**: 텍스트가 그리드를 넘나들거나 압도적인 크기로 배치되어 역동적이고 지루하지 않습니다.

### ⚡ 인터랙션 요소 (Interaction)
*   **Kinetic Typography**: 텍스트가 회전하거나, 스크롤에 따라 변형되는 등 글자 자체가 디자인 요소로 움직임.
*   **Marquee Effect**: 끊임없이 흐르는 텍스트 배너가 속도감과 긴급함(Urgency)을 부여합니다.
*   **Smooth Scroll**: 관성이 적용된 스크롤로 묵직하고 부드러운 조작감을 제공합니다.

### 🌈 컬러 시스템
| 구분 | 색상(Color) | Hex Code (추정) | 비고 |
| :--- | :--- | :--- | :--- |
| **Primary** | **Lime Green** | `#B1FF00` | 형광 톤의 강렬한 에너지, 디지털 감성 |
| **Text** | **Off-White** | `#EAEAEA` | 블랙 배경 위에서의 높은 가독성 |
| **Background** | **Deep Black** | `#000000` | 몰입감, Primary 컬러 강조 |

## 3. Funtown Studio
**URL**: https://www.funtownstudio.com/

### 🎨 UX/UI 강점
*   **Playful & Creative**: 둥근 형태의 서체와 도트 그리드(Dot Grid) 배경이 마치 놀이터나 캔버스를 연상시킵니다.
*   **Character-driven**: 귀여운 캐릭터 요소들을 곳곳에 배치하여 사용자에게 친근감과 즐거움을 줍니다.

### ⚡ 인터랙션 요소 (Interaction)
*   **Surprise Elements**: 클릭할 때마다 캐릭터가 바뀌거나 요소가 튀어나오는 등 예상치 못한 "재미(Fun)"를 줍니다.
*   **Elastic Hovers**: 버튼이나 메뉴에 마우스를 올리면 젤리처럼 튕기거나(Bouncy) 부드럽게 확장되는 탄성 있는 애니메이션을 사용합니다.

### 🌈 컬러 시스템
| 구분 | 색상(Color) | Hex Code (추정) | 비고 |
| :--- | :--- | :--- | :--- |
| **Primary** | **Vivid Orange** | `#FF6B44` | 활기, 창의성, 즐거움 |
| **Background** | **Light Grey** | `#F1F1F1` | (Dot Pattern 포함) 눈이 편안한 밝은 배경 |
| **Accents** | **Pastel Tones** | Pink, Blue, Green | 캐릭터 및 장식 요소에 다양하게 사용 |
