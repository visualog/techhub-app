# TechBird 디자인 시스템 - 컬러 가이드

TechBird의 리브랜딩에 따라 기존의 인디고(Indigo) 테마에서 신뢰감 있고 시원한 **블루(Blue) 테마**로 전환되었습니다.

## 1. 브랜드 핵심 컬러 (Brand Logic)

모든 브랜드 포인트 컬러는 Tailwind의 `blue` 계열로 일원화되었습니다.

| 요소 | 라이트 모드 | 다크 모드 | 비고 |
| :--- | :--- | :--- | :--- |
| **Primary (기준색)** | `blue-600` | `blue-400` | 전체적인 시각적 강조 |
| **Accent (강조색)** | `blue-500` | `blue-300` | 호버 효과 및 중요 알림 |
| **Background (배경색)** | `bg-white` | `bg-slate-950` | 깨끗하고 깊이 있는 배경 |

---

## 2. 주요 구성 요소 상세 컬러

### 🌊 헤더 스크롤 진행 바 (Scroll Progress)
사용자의 가독 흐름을 돕는 헤더 하단의 진행 상태 표시줄입니다.
- **라이트 모드:** `bg-blue-600`
- **다크 모드:** `bg-blue-400`

### 🏷 아티클 카테고리 칩 (Category Chips)
기사 카드 하단에서 카테고리를 강조하는 칩의 색상입니다.
- **라이트 모드:**
  - 배경: `bg-blue-100`
  - 텍스트: `text-blue-800`
- **다크 모드:**
  - 배경: `bg-blue-900`
  - 텍스트: `text-blue-200`

### 🔍 검색 및 입력 요소
- **포커스 링 (Focus Ring):** `ring-blue-500`
- **검색 아이콘:** `text-blue-500`

### ✨ 기타 디자인 요소
- **404 페이지 그라데이션:** `from-blue-600 to-cyan-500`
- **토스트 알림 (Info):** `border-blue-500`, `text-blue-500`
- **북마크 버튼 활성화:** `text-blue-500`

---

## 3. 시스템 설정 값 (globals.css)

최하단 시스템 레벨에서도 파란색 테마가 기본값이 되도록 CSS 변수가 업데이트되었습니다.

```css
:root {
  --primary: 221.2 83.2% 53.3%;  /* Blue 600 계열 */
  --ring: 221.2 83.2% 53.3%;
}

.dark {
  --primary: 217.2 91.2% 59.8%; /* Blue 400 계열 */
  --ring: 224.3 76.3% 48%;
}
```
